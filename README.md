## Description

This is an implementation from the IBM Quantum code challenge, using "strict" Hexagonal Architecture and TDD.

More details at the [development notes](docs/DEV-NOTES.md).

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
